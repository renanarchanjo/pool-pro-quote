# SIMULAPOOL - Brand Kit Oficial

**Sistema profissional de orçamentos automatizados para piscinas de fibra**

---

## 🎯 Sobre a Marca

SIMULAPOOL é uma startup brasileira de tecnologia especializada no mercado de piscinas. Oferece um sistema SaaS moderno que automatiza completamente o processo de orçamento para lojistas, combinando simplicidade, velocidade e profissionalismo.

### Posicionamento
- **Público-alvo**: Lojistas e vendedores de piscinas de fibra no Brasil
- **Proposta de valor**: Orçamentos automatizados, rápidos e profissionais
- **Diferencial**: Tecnologia acessível + interface premium + resultados instantâneos

## 🔤 Tipografia

### Fonte Display (Títulos)
**Montserrat**
- Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold), 800 (ExtraBold)
- Uso: Títulos (H1-H6), CTAs, navegação, destaques
- Características: Geométrica, moderna, alta legibilidade

```css
font-family: 'Montserrat', sans-serif;
```

### Fonte Corpo (Textos)
**Inter**
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
- Uso: Parágrafos, descrições, formulários, labels, UI
- Características: Humanista, legível em telas, otimizada para interfaces

```css
font-family: 'Inter', sans-serif;
```

### Hierarquia Tipográfica

```
H1: Montserrat Bold 48-72px (Hero titles)
H2: Montserrat Bold 36-48px (Section titles)
H3: Montserrat Bold 24-32px (Card titles)
H4: Montserrat SemiBold 20-24px (Subtitles)
Body Large: Inter Regular 18-20px (Descriptions)
Body: Inter Regular 16px (Content)
Small: Inter Regular 14px (Captions, labels)
Tiny: Inter Regular 12px (Fine print)
```

## ✨ Elementos Visuais

### Ondas Minimalistas
Linhas fluidas e suaves representando água em movimento. Usar com transparência sutil (10-20%) como textura de fundo.

### Bordas Arredondadas
- Cards: 12-16px
- Botões: 8-12px
- Inputs: 8px
- Modais: 16-24px

### Ícones
Estilo: Line icons, stroke 2px
Biblioteca recomendada: Lucide Icons
Cor: Pool Blue ou Deep Blue

### Sombras

#### Shadow Card
```css
box-shadow: 0 10px 40px -10px rgba(14, 165, 233, 0.25);
```

#### Shadow Elegant
```css
box-shadow: 0 20px 60px -15px rgba(14, 165, 233, 0.35),
            0 0 0 1px rgba(214, 229, 245, 1);
```

#### Shadow Pool
```css
box-shadow: 0 25px 80px -20px rgba(14, 165, 233, 0.4),
            0 10px 30px -10px rgba(6, 182, 212, 0.2);
```

### Animações
- Fade in: 500ms ease-out
- Slide in: 300ms ease-out
- Wave motion: 3s ease-in-out infinite
- Hover scale: 110% com transition 200ms
